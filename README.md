# 读墨 · macOS 概念版 / Dumo for macOS (UI concept)

把「**微信读书笔记自动同步到 Obsidian**」设想成一个常驻 Mac 菜单栏的后台 App —— 一份高保真 UI 原型。

🔗 **在线预览**：https://hannahlovegood.github.io/dumo-macos/

## 设计亮点

- 🖥️ 仿 macOS **原生窗口控件**与**菜单栏小图标**交互（点一下「墨」试试）
- 🔄 「自动同步到 Obsidian」「有新笔记」等后台同步概念
- 📖 书库 / 书架 / 偏好设置 等完整应用界面，用真实书目（如《置身事内》）做样例
- 💡 产品理念藏在细节里：「Mac 用户大部分时候不会打开它，它默默在背景工作」

## 技术

单文件 HTML 概念原型，专注交互与视觉表达。

> 实际能用的导出工具见 web 版：https://github.com/hannahlovegood/dumo-web
